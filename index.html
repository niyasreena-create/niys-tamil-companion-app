GitHub Pages மூலமாக லைவ் வெப்சைட்டாக மாற்றுவது எப்படி?

மிக எளிதாக 1 நிமிடத்தில் ஆப்பை இணையத்தில் பதிவேற்றலாம்

1

முதலில் உங்கள் GitHub கணக்கிற்குள் லாக்-இன் செய்து, ஒரு புதிய Repository-ஐ உருவாக்கவும். அதற்கு niyas-tamil-companion-app என்று பெயரிடவும்.

2

அங்கே ஒரு புதிய கோப்பை உருவாக்கி அதற்கு index.html என்று பெயரிட்டு, இந்த எடிட்டரில் இருக்கும் முழு குறியீட்டையும் (Code) அப்படியே காப்பி செய்து பேஸ்ட் செய்து Commit செய்யவும்.

3

இப்போது உங்கள் ரெபாசிட்டரியின் Settings -> Pages என்ற பகுதிக்குச் செல்லவும். அங்கே Source என்பதில் Deploy from a branch என்பதைத் தேர்வு செய்து, Branch பிரிவில் main அல்லது master என்பதை தேர்வு செய்து சேமிக்கவும்.

வெற்றிகரமாக முடிந்தது! இப்போது 1 நிமிடத்திற்குள் உங்களுக்கு ஒரு பிரத்யேக இணையதள முகவரி (லிங்க்) கிடைத்துவிடும். அதை உங்கள் நண்பர்களுடனும், சமூக வலைத்தளங்களிலும் பகிர்ந்து கொள்ளலாம்!
